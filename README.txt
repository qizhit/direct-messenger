This program have five main functions:
1. a simple file management tool which can output the contents of a specified directory with their absolute path and create, delete, or read a file with "dsu" extension.
2. a simple jounal system that users can edit or check their profile and posts.
3. a simple system that allows users to publish their posts or bio online.
4. a simple system that allows users to retrieve data from OpenWeather and LastFM API, and to use @KEYWORD to get one data.
5. a simple system that allows users to send messages to another user or retrieve messages from other users.

To use it, You can just run the code and follow the prompts.
